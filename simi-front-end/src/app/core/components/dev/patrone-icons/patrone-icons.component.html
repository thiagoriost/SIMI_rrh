ICONOS

Material cuenta con un componente para usar iconos mat-icon, el cual usa la libreria Material Icons Library.

Adicionalmente se cuenta con una libreria de iconos personalizados SIMI Icons Library, la cual se puede usar con el mismo componente mat-icon de la siguiente manera:

Codigo
  <mat-icon fontSet="simi" fontIcon="icon-simi-ver_revisiones"></mat-icon>
Resultado

Donde fontSet="simi" indica que se usara la libreria de iconos personalizados y fontIcon="icon-simi-ver_revisiones" indica el nombre del icono a usar.
